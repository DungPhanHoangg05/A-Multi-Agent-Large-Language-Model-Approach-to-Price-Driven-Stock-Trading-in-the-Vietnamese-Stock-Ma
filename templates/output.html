(avg − μ_lô) / (σ_lô + ε)

< −1.5 = tin bất thường tiêu cực.
Dùng phát hiện tin bất thường: sign(SENT_ZSCORE) × tanh(|SENT_ZSCORE|) × kỹ_thuật
REL_SENTIMENT
{{ "%+.4f"|format(sn.rel_sentiment) }}
avg_target − avg_ngành

> 0 = cổ phiếu outperform ngành về tin tức.
< 0 = cổ phiếu underperform ngành.
